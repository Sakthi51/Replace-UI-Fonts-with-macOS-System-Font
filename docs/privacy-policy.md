# Privacy Policy — Glyph

**Last updated:** July 2026

## Summary

Glyph does not collect, store, transmit, or share any user data.

## Data Collection

This extension collects **no data whatsoever**. Specifically:

- No browsing history
- No personal information
- No analytics or telemetry
- No cookies or authentication tokens
- No page content or form data
- No network requests to external servers

## Data Storage

The only data stored is your preferences (which sites you've enabled the font override on). This is stored locally in Chrome's `chrome.storage.sync` and synced only to your own Chrome profile via your Google account. The extension developer has no access to this data.

## Permissions Explained

| Permission | Why |
|---|---|
| `storage` | Save your list of enabled sites |
| `activeTab` | Read the current tab's URL to show the site toggle |
| `scripting` | Inject the font-override script into pages you've enabled |

## Third Parties

This extension communicates with no third-party services.

## Open Source

The source code is fully inspectable. The extension contains only client-side JavaScript that modifies font rendering on the pages you choose.

## Contact

For questions about this privacy policy, open an issue on the GitHub repository.
